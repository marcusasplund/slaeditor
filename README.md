## installation

````bash
    $ git clone https://github.com/marcusasplund/slaeditor.git

    $ cd slaeditor

    $ yarn

    $ yarn start
````

This will open application at http://localhost:3000/ in browser

## build a release

````bash
    $ yarn build

````
This will generate a release directory with your minified/rev'd assets.
